# 🩺 Disease Progression Explorer

An interactive R Shiny dashboard for exploring Chronic Kidney Disease progression using clinical data visualization.

## 🚀 Live Demo

[https://<your-posit-connect-url>](https://019f08eb-a806-194c-1d02-d2976d9c83fd.share.connect.posit.cloud/)

## 📊 Features

- Interactive age filter
- Diabetes & hypertension filters
- KPI cards
- Age distribution
- Kidney function scatter plot
- Creatinine distribution
- Patient records table

## 🛠 Tech Stack

- R
- Shiny
- ggplot2
- Plotly
- dplyr
- DT
- bslib
